# Claims Requiring Verification & Audit Registry

This registry tracks all product capabilities, performance statistics, compliance claims, and integrations used across Asif Digital landing pages. 

Each capability is assigned one of the following audit statuses:
- **Status A (Formally Verified):** The business owner has verified or certified this claim (Asif Digital possesses direct documentation).
- **Status B (Tech Capability - Uncertified):** True of our standard tech stack, but not officially certified (we build it, but no formal vendor partnership exists).
- **Status C (Illustrative Scenario):** Mockup, demonstration, or example workflow (clearly labeled in UI).
- **Status D (Unverified / Removed):** Requires client verification / removed or replaced with safe copy.
